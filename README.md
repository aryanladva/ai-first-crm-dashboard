# 🚀 AI CRM Dashboard

This is an AI-powered CRM dashboard built using FastAPI, React, and LangGraph.

---

## 🔹 Features

- 📋 Log doctor interactions
- 🤖 AI-based sentiment analysis (Positive / Negative / Neutral)
- 🔔 Follow-up suggestions
- 📊 Graph analytics (Recharts)
- 🛠 Tool-based workflow using LangGraph
- 📁 Interaction history tracking

---

## 🔹 Tech Stack

- Backend: FastAPI (Python)
- Frontend: React.js
- AI Workflow: LangGraph
- Charts: Recharts

---

## 🔹 How to Run

### Backend
cd backend  
python -m uvicorn main:app --reload  

### Frontend
cd frontend  
npm start
